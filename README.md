# NetBannerNG

Open source alternative to Microsoft NetBanner. Fully compatible with current NetBanner GPOs.

It would include alternative ADMX files for extended user cases.

## Requirements
It's based on .NET Framework 4.0 to allow Win 7 and upper operating systems with minimal dependency.

Netbanner GPOs are required to manage the classifications.
